# Amedicase Design System - Component Inventory

## Overview
Lista completă a tuturor secțiunilor și elementelor de pe pagină, structurată ca un design system pentru referință și reutilizare.

## 1. HeaderSection
**Descriere:** Header-ul principal al site-ului, fixat în partea de sus, conține logo, navigare și buton CTA.

### Elemente:
- **HeaderLogoMobile** - Logo-ul pe mobile (180px width)
- **HeaderLogoTablet** - Logo-ul pe tablet (261px width)
- **HeaderLogoDesktop** - Logo-ul pe desktop (260.98px width)
- **HeaderBurgerMenuButton** - Butonul hamburger pentru mobile
- **HeaderBurgerMenuIcon** - Iconița hamburger (3 linii)
- **HeaderCloseMenuIcon** - Iconița X pentru închiderea meniului
- **HeaderMobileMenuOverlay** - Overlay-ul pentru meniul mobile
- **HeaderMobileMenuBackdrop** - Backdrop blur pentru overlay
- **HeaderNavigationLinks** - Lista de linkuri de navigare

- **HeaderBookCallButtonMobile** - Butonul "Book a Call" pe mobile

- **HeaderBookCallButtonDesktop** - Butonul "Book a Call" pe desktop
- **HeaderNavigationContainerDesktop** - Container-ul de navigare pe desktop (backdrop blur)

## Usage Examples

### Example 1: Reuse Hero Button
```
"Ia **HeroButtonDesktop** și pune-l în **OutsourcingSolutionsSection** ca buton secundar"
```

### Example 2: Reuse Card Component
```
"Folosește **CardServiceItem** în secțiunea **HowItWorksSection** la poziția 3"
```

### Example 3: Reuse Title Component
```
"Refolosește **GradientTitle** în noua secțiune Blog cu textul 'Latest Articles'"
```

### Example 4: Clone Component
```
"Clonează **TeamMemberCard** și creează **TestimonialCard** pentru secțiunea Testimonials"
```

### Example 5: Reuse Form Field
```
"Folosește **ContactFormFieldEmail** în formularul de newsletter din Footer"
```

---

## Notes
- Toate componentele sunt responsive și au variante pentru mobile, tablet și desktop
- Numele componentelor sunt standardizate pentru ușurința referinței
- Fiecare element poate fi reutilizat sau clonat în alte secțiuni
- Componentele shared pot fi folosite în orice secțiune nouă
